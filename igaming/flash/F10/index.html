實測心得：這款是「初見驚豔、玩久單調」的典型。以下從體感 → 原因 → 企劃學到的點拆開講。
面向 1｜體感：兩個期待點、一個名場面，但玩久會膩

觀察給分大方，基礎連消幾次就有不錯分數；可期待的只有 X2 符號（多個相加）與 警鈴（滾 ×5–×50），基本遊戲就會出現。最期待的名場面＝多個 X2 被警鈴一次替換成更高倍數，初見很驚豔。

反面但大多只偶爾掉一個 X2、警鈴很少來，畫面常退回單純消消樂；FREE GAME 快一小時都測不到一次。

面向 2｜原因：固定 RTP 的分配取捨

觀察每款遊戲的總返還率是固定的（約 96%），等於有一筆固定的獎金預算要發：這筆錢要嘛發給「常常中的小獎」、要嘛留給「難得的大倍數」，只能二選一。這款把大部分預算拿去讓基本遊戲「中得多、給得大」，留給倍數的就很少——所以倍數上限只有 ×50、出現也不頻繁。

觀察外觀學宙斯／賽特（消除撒倍數），數字沒學到——宙斯平常摳、倍數上看 ×100–×500 才刺激；這款基礎太大、天花板太低、倍數太少，所以平。

面向 3｜大獎不是鎖在免費遊戲，而是「條件太苛」

觀察大獎要同時「多個 X2 ＋ 警鈴 ＋ 高倍 ＋ 有中獎」才成立，基本遊戲也可能發生、只是很少；X2 每一轉結束就清空、又不常出現，平常很難疊多。

觀察免費遊戲難進，是因為「FG 倍數不重置、一路累加」太強 → 只能用低觸發率平衡（強功能配低頻率）。

面向 4｜企劃角度學到的（解方方向）

學習幾個可帶走的設計點：

  • 「一次質變」的爆點：警鈴把盤面一堆 x2 一次掃成 ×50，「累積的小東西瞬間全升級」情緒很強，可移植到別的機制（收集到一半一次全升、小倍數一次進化）。
  • 強功能配低頻率：FG 倍數不重置很強 → 觸發壓低；「愈強愈稀有」方向對，只是這款做過頭。
  • 好點子要餵到「看得到」：替換機制雖好，但要四件事同時才發光 → 幾乎看不到；要出現得夠頻繁、倍數夠大，玩家才感受得到。
  • 基本遊戲不能只當「等大獎的走廊」：爽點條件太苛、又鎖一部分在 FG，基本遊戲沒有等價替代刺激 → 只剩消消樂 → 膩；基本遊戲要能自己撐住娛樂性。
