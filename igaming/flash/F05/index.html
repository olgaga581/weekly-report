遊戲玩法
  1. Cluster Pays：3+ 同符號水平或垂直相連即得分，不限位置。
  2. Prize Symbol 雙啟動：中獎位置第 1 次 marked，第 2 次變 Prize 分數球（0.2x~8x bet）。
  3. Prize Upgrade：Prize 結算前隨機升級為 Gold（10x–50x）或 Platinum（100x–500x）。
  4. Free Game 累積：marks + Prize symbols 整個 FG 全程保留，跨 spin 累積。
  5. 觸發：3 SCATTER = 10 spins；每多 1 SCATTER +2 spins，FG 內可 retrigger。
  6. 賠率表（Paytable）：各符號連線賠率與特殊符號一覽。

期待點：分數球出現 → 升級 Gold／Platinum 大球的那條直接路徑。

類 Cash on Reel 的分數球收集，期待點專注、獲分路徑直接。以下分四個面向，各列觀察與可能解方。
面向 1｜機制體驗

觀察有條件獲得分數球、再隨機升級為大球，期待點專注、獲分路徑直接，base game 就能拿到不錯分數，機制蠻有趣。

面向 2｜Free Game 進入頻率

觀察FG 是 base game 的強化，會 hold 住前面的分數球可反覆拿，最佳盤面是 4×4 全分數球；但進 FG 不太容易，可能因為這個強化太強，只能降低進入頻率，有點可惜。

面向 3｜Free Game 變化性

觀察其實 base game 就可能玩到跟 FG 一樣的盤面（但不能反覆拿分數球），FG 的獨特感不夠。

解方FG 放大盤面（5×5、6×6），把天花板撐高，也讓進 FG 更有特別感。

面向 4｜升級演出與瞇牌感

觀察Gold／Platinum 是「結算前隨機升級」，玩家沒有決策、也可能沒有累積過程，升級變成純結果通知，瞇牌感容易流失。

解方讓連消次數／球的數量影響升級機率，給玩家一個「再連一次說不定升級」的理由（同時取代現在的純隨機升級）。
